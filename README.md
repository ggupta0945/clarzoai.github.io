# Clarzo.ai — Website

Static marketing and product website for **Clarzo.ai**, an AI-powered wealth intelligence platform.

Live site: https://ggupta0945.github.io/clarzoai.github.io/

## Pages

| File | Description |
|------|-------------|
| `index.html` / `Clarzo_Landing_Page.html` | Home page — hero, features, FAQ, CTA |
| `Clarzo_Product_Page.html` | Product deep-dive with feature showcases |
| `Clarzo_Stock_Discovery.html` | Sector & company discovery with CompanyGPT mock |
| `Clarzo_Knowledge_Repo.html` | Articles, guides, resources, and calculators index |
| `Clarzo_Calculators.html` | Financial calculators (EMI, SIP, Tax, PPF, FD, etc.) |
| `Clarzo_About_Us.html` | Mission, story, team, and values |
| `Clarzo_Privacy_Security.html` | Privacy policy and security practices |
| `Clarzo_Terms.html` | Terms & conditions |

## Stack

- Pure HTML + CSS + vanilla JS — no build step, no dependencies
- Fonts via Google Fonts (DM Sans + Playfair Display)
- Hosted on **GitHub Pages**

## Local preview

Just open `index.html` in a browser, or run a quick static server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment

Pushes to the default branch are served automatically by GitHub Pages.
Configure under **Settings → Pages**.
